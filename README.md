# Test Site 🚀

A simple Node.js and Express web server setup.

## Project Structure

- **server.js.js** - Main Express server file
- **public/** - Static files (CSS, images, JavaScript)
- **views/** - HTML templates

## Setup

1. Install dependencies:

```bash
npm install
```

2. Start the server:

```bash
npm start
```

3. Open your browser and navigate to `http://localhost:3000` to see the test site in action.

## About

This project is a basic template for a Node.js and Express web application. It serves static files from the `public` directory and renders HTML templates from the `views` directory.
